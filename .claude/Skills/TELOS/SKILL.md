# TELOS - Life Operating System

TELOS (Telic Evolution and Life Operating System) is a dual-context system for managing personal life context and project analysis.

## Workflow Routing

| Action | Trigger | Workflow |
|--------|---------|----------|
| **Update TELOS** | "add to TELOS", "update my goals", "add belief" | Workflows/Update.md |
| **View TELOS** | "show my TELOS", "what are my goals", "my beliefs" | Direct file read |
| **Project Analysis** | "analyze project TELOS", "project goals" | Direct analysis |

## Personal TELOS Framework

All personal TELOS files reside in USER/ subdirectory:

### Core Philosophy
- MISSION.md - Your life mission and purpose
- BELIEFS.md - Core beliefs and values
- GOALS.md - Current goals and aspirations
- WISDOM.md - Accumulated wisdom and insights

### Life Data
- BOOKS.md - Impactful books and takeaways
- MOVIES.md - Films that shaped perspective
- LEARNED.md - Key lessons learned
- WRONG.md - Things you were wrong about

### Mental Models
- FRAMES.md - Mental frames for decision-making
- MODELS.md - Mental models used
- NARRATIVES.md - Stories that guide you
- STRATEGIES.md - Life strategies

### Goals & Challenges
- PROJECTS.md - Active projects
- PROBLEMS.md - Problems being worked on
- CHALLENGES.md - Current challenges
- PREDICTIONS.md - Predictions and bets

### Change Tracking
- updates.md - Comprehensive changelog of all TELOS updates

## Working with Personal TELOS

### Reading TELOS
Simply read the relevant files directly from USER/ directory.

### Updating TELOS
**ALWAYS use the Update workflow** to ensure:
- Timestamped backups before changes
- Change logging in updates.md
- Version history preservation

## Security & Privacy

**Personal TELOS contains highly sensitive personal information:**
- NEVER commit to public repositories
- NEVER share publicly
- ALWAYS backup before changes
