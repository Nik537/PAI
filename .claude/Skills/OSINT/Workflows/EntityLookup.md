# Entity Lookup Workflow

For domains, IPs, file hashes, and other technical indicators.

## Phase 1: Basic Resolution
- DNS records
- WHOIS data
- IP geolocation
- ASN information

## Phase 2: Infrastructure Mapping
- Subdomains
- Related domains
- Reverse DNS
- Certificate transparency

## Phase 3: Reputation Check
- Threat feed presence
- Blocklist status
- Abuse reports
- Malware associations

## Phase 4: Historical Analysis
- Passive DNS history
- Wayback Machine
- Previous owners
- Registration changes

## Phase 5: Technical Fingerprinting
- Technologies in use
- Server software
- Security headers
- Open ports

## Phase 6: Content Analysis
- Hosted content
- Similar sites
- Linked resources
- Associated files

## Phase 7: Network Analysis
- Related infrastructure
- Common hosting
- Shared certificates
- Traffic patterns

## Phase 8: Attribution
- Potential operators
- Geographic indicators
- Language analysis
- Timing patterns

## Phase 9: Synthesis
- Threat assessment
- Confidence rating
- Recommended actions
- IOC export
