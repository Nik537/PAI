# Full Council Debate Workflow

## Process

### Step 1: Frame the Question
Clearly articulate the topic for debate. Ask user for clarification if needed.

### Step 2: Invoke Council Members
Launch 5 parallel agents (using Task tool), each embodying one council role:
- Optimist, Skeptic, Pragmatist, Visionary, Historian

Each agent receives:
- The debate topic
- Their role and perspective
- Instructions for Round 1 statement

### Step 3: Collect Round 1
Gather all opening statements. Format and present to user.

### Step 4: Execute Round 2
Share all Round 1 statements with each agent.
Each responds to 2-3 points from other agents.

### Step 5: Synthesize
Combine all perspectives into synthesis:
- Identify agreement clusters
- Note persistent disagreements
- Extract novel insights
- Formulate recommended action

### Step 6: Present Results
Format complete debate transcript with synthesis.
